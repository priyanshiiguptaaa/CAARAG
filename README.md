# Confidence-Aware Adaptive Retrieval-Augmented Generation (RAG)

> **Journal Publication Project**

## Overview

This repository implements a full **Confidence-Aware Adaptive RAG** system from scratch, designed for journal-level reproducibility and ablation studies.

The system dynamically adjusts the number of retrieval rounds (and the retrieved context) based on a **confidence score** assigned to each generated answer, avoiding unnecessary LLM calls when the model is already confident, and triggering targeted re-retrieval when it is not.

## Quick Start

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

### 2. Run Step 1 — Corpus Preparation

```bash
cd confidence_aware_rag
python step1_corpus_preparation.py
```

This will:
- Download HotpotQA (~600MB, cached by HuggingFace)
- Extract ~300 k unique Wikipedia passages
- Chunk them into 150-word windows
- Apply lightweight preprocessing
- Extract keywords per chunk (KeyBERT)
- Save `data/corpus.json` and `data/corpus_stats.json`

Expected runtime: **5–15 minutes** depending on hardware.

## Configuration

All parameters live in `configs/config.yaml`. Key knobs:

| Key | Default | Description |
|-----|---------|-------------|
| `corpus.chunk_size` | 150 | Words per chunk |
| `corpus.overlap` | 0 | Sliding-window overlap |
| `corpus.add_keywords` | true | KeyBERT extraction |
| `dataset.max_train_samples` | null | Limit for fast debugging |
| `confidence.threshold_low` | 0.4 | Re-retrieve below this |
| `confidence.threshold_high` | 0.75 | Accept directly above this |
